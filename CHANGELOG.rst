[0.6.0] - 2019-09-18
--------------------

Added
^^^^^

- Added ``dmaudit report`` subcommand (previously this was the functionality in

- Added ``dmaudit mimetype`` subcommand to be able to test what mimetype

Changed
^^^^^^^

- Command line interface now takes subcommands
